## 尾调用

尾调用（Tail Call）是函数式编程的一个重要概念, 就是指某个函数的最后一步是**调用另一个函数**。

``` js
//不是尾调用
function fact(n) {
    if (n <= 0) {
        return 1;
    } else {
        return n * fact(n - 1);
    }
}

//尾调用，尾递归
function fact(n, r) {
    if (n <= 0) {
        return 1 * r;
    } else {
        return fact(n - 1, r * n);
    }
}
```

## 尾递归：相对节省内存。

**注意:** 尾递归函数依然还是递归函数，如果不优化依然跟普通递归函数一样会爆栈，该展开多少层依旧是展开多少层。不会爆栈是因为语言的编译器或者解释器所做了“尾递归优化”，才让它不会爆栈的。
